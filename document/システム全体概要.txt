==========================================
JHMS（就職活動管理システム）全体概要
==========================================

【システム概要】
就職活動管理システム（JHMS）は、大学の就職支援室が学生の就職活動を効率的に管理するためのWebアプリケーションです。

【システム構成】

1. フロントエンド
   - HTML/CSS/JavaScript
   - JSP（JavaServer Pages）
   - jQuery

2. バックエンド
   - Java Servlet
   - JSP
   - JDBC（データベース接続）

3. データベース
   - MySQL（想定）

4. Webサーバー
   - Apache Tomcat 10.1.19

【ユーザー権限】

1. 教師権限
   - 学生管理
   - 企業管理
   - 書類提出チェック
   - 試験・面接情報登録
   - 選考段階管理

2. 学生権限
   - 就職活動状況確認
   - 書類提出状況確認

【主要機能一覧】

1. 認証・認可機能
   - ログイン機能
   - ログアウト機能
   - 権限チェック機能

2. 学生管理機能
   - 学生情報の登録・編集・削除
   - 学生検索機能
   - 学生一覧表示

3. 企業管理機能
   - 企業情報の登録・編集・削除
   - 企業検索機能
   - 企業一覧表示

4. 書類管理機能
   - 書類提出状況の確認
   - 書類確認・コメント機能
   - 書類提出状況の検索・フィルター

5. 選考管理機能
   - 試験・面接情報の登録
   - 選考段階の管理
   - 選考結果の記録

【画面構成】

1. ログイン画面（login.html）
2. ダッシュボード画面（DashBoard.jsp）
3. 学生管理画面（StudentManagement.jsp）
4. 企業管理画面（companyManagement.jsp）
5. 書類提出チェック画面（documentCheck.jsp）
6. 試験・面接情報登録画面（InterviewExamInput.jsp）
7. 選考段階管理画面（SelectionStage.jsp）

【セキュリティ機能】

1. 認証機能
   - ユーザーID・パスワード認証
   - セッション管理

2. 認可機能
   - 権限ベースのアクセス制御
   - 画面・機能の権限チェック

3. セキュリティ対策
   - SQLインジェクション対策
   - セッション固定攻撃対策
   - XSS対策

【技術仕様】

1. 開発言語
   - Java 8以上
   - HTML5
   - CSS3
   - JavaScript

2. フレームワーク・ライブラリ
   - Servlet API
   - JSP
   - JDBC
   - jQuery

3. データベース
   - MySQL 5.7以上

4. Webサーバー
   - Apache Tomcat 10.1.19

【ファイル構成】

src/main/
├── java/
│   ├── beans/          # JavaBeans
│   ├── controller/     # コントローラー
│   ├── dao/           # データアクセスオブジェクト
│   ├── dto/           # データ転送オブジェクト
│   ├── service/       # ビジネスロジック
│   ├── servlet/       # サーブレット
│   └── utils/         # ユーティリティ
└── webapp/
    ├── css/           # スタイルシート
    ├── images/        # 画像ファイル
    ├── js/            # JavaScript
    └── WEB-INF/
        ├── jsp/       # JSPファイル
        └── web.xml    # Webアプリケーション設定
